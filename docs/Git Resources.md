## How to Submit Pull Requests
- [ ] Fork the target repo
![[github-fork.png]]
- [ ] Clone _your fork_ to your computer
![[kraken-clone.png]]
- [ ] Create a feature branch
![[kraken-branch.png]]
- [ ] Make your changes
- [ ] Stage your changes
![[kraken-stage.png]]
- [ ] Commit your changes
![[kraken-commit.png]]
- [ ] Push your changes to your fork
![[kraken-push.png]]
- [ ] Open a Pull Request to the original repo
![[github-pr-1.png]]
![[github-pr-2.png]]
